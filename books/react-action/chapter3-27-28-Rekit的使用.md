# 27/28 Rekit的使用

Rekit是作者创建的一个Github开源项目，可以让你在浏览器端（Rekit Studio)来进行编码。

它提供了组件的可视化创建，帮你用最佳实践的方式组织好代码，对模块的改动，它能帮你将所有关联的进行修改。


总体来说，作者还是很强的，不一定要使用它，但是它所生成的代码可以当做最佳实践的参考。

另外，感兴趣可以关注下它的实现。
