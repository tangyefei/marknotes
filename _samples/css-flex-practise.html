        此处介绍了如何让footer始终在网页底部（哪怕中间内容不够多）：

        1. 高度设置为 min-height: 100vh 撑满
        2. 内容区域使用掉 flex: 1 
        3. 那么对应footer使用的高度都是在最底部了
      
        .Site {
          background: #000;
          display: flex;
          min-height: 100vh;
          flex-direction: column;
          color: #FFF;
        }

        .Site-content {
          flex: 1;
        }
