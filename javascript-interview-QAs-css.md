#1.盒子模型

- 盒子模型包括`content+padding+border+margin`
- 盒子模型的宽高收到`box-sizing`属性的影响
	- 设置为`box-sizing:content-box`是`content`区域的大小，通常为默认
	- 设置为`box-sizing:border-box`是`content+padding+border`的大小，低版本IE的默认
